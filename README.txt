Detta projekt är baserat på den faktiska, romerska "dagstidningen" Acta Diurna. Jag valde att digitalisera den med det 
latinska hitte-på ordet "Interette", som ska betyda "Internet". Tanken är att fånga upp den romerska vardagen i formen av en modern click-bait nyhetssida. 
Jag tog inspiration från olika dagblad och gick efter extra click-bait inriktade rubriker, artiklar och rykten. 

Bifogat i repot är en desktop-och mobilskärmdump (mobile.png och desktop.png)

Färgerna var rätt så enkla att bestämma eftersom det finns historiskt material att inspireras av. Pergament användes av romarna, 
så dess färg användes som bakgrund. Purpurröd och guld passar också den romerska stilen. Typsnittet känns väldigt romerskt och en specifik
font användes som konverterar alla "u" till "v" (romarna skrev "u" som "v". Vad som angår layout så valde jag att använda grid för 
att få nyheterna att se ut som "nyhetskort" medan flex användes för positionering. 

För css mönstrena så höll jag det rätt så enkelt. Hover används på länkar och artiklar för en visuell återkoppling. Details/summary användes för 
FAQ delen (accordion). 

Kända begränsningar:

Ingen skip länk
Ingen avancerad meny
Inga css-variabler används 

Förbättringsidéer:

Lägga till css-variabler
Skapa skip länk 
T.ex implementera en hamburgermeny
Dark-mode (pergamentfärgen kan vara bländande)
